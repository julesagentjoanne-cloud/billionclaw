# startupClaw - Product Integration - Creative Agent

## 1. Agent / capability
- Creative Agent

## 2. User-facing job
- generate structured creative concepts and test packs based on pain points, winners, and market signals

## 3. Primary UI surfaces
- report section: creative opportunities
- agent card: Creative Agent
- workflow step: generate creative system
- asset/output screen: hook library, creative matrix, test pack
- review screen: creative performance and refresh recommendations

## 4. User flow impact
- appears when creative quality, variation, or refresh is the main bottleneck
- gives the founder a visible path from pain insights to launch-ready creative tests

## 5. Key UI states
- not started
- pain angles mapped
- hooks generated
- shortlist ready
- creative pack ready
- linked to campaign review
- blocked / missing brand assets

## 6. Visible outputs
- pain-angle buckets
- shortlisted hooks
- creative concepts by format/persona
- landing-page alignment notes
- prioritized test pack

## 7. Connected data/signals
- customer pain points
- competitor ad intelligence
- prior winners
- brand/style assets
- landing page context

## 8. Cross-agent connections
- takes performance context from Kill-Scale Agent
- feeds assets into GTM Execution Agent and Content Engine Agent
- uses Messaging Agent positioning inputs

## 9. MVP version
- one Creative Agent screen showing angle buckets, top hooks, and exportable creative test pack

## 10. Later expansion
- editable creative workspace
- direct asset generation previews
- live feedback loop from campaign data
