# startupClaw - Buyable Agent Catalog

## Purpose
Define the execution agents a founder can select after receiving the startupClaw report.

## Simple overview
The founder should be able to buy help in the form of concrete execution modules.

All startupClaw agents should follow the [[Expert Agent Standard]]:
- they should behave like top performers in their field
- they should produce output strong enough that a founder would pay for it
- they should clearly fulfill the job they were hired for

## Agents

## Modeling rule
Each agent should be modeled like a venture-studio expert:
- with a clear domain
- with a clear job to be done
- with a visible skill stack inside that domain

Examples:
- GTM Execution Agent -> prospect discovery, outreach sequencing, message crafting, channel testing
- Creative Agent -> pain-angle mapping, hook writing, layout direction, winner cloning
- Community Agent -> motivation mapping, ritual design, champion systems, feedback routing
- Content Engine Agent -> hook research, voice adaptation, asset packaging, scheduled publishing

## Product rule
The founder should see:
- what each agent is for
- what each agent produces
- which one startupClaw recommends next
- why that agent is the fastest route to progress
