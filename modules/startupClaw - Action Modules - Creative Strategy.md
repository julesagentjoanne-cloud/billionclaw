# startupClaw - Action Modules - Creative Strategy

## Purpose
Define the creative-generation action modules startupClaw can execute when a startup needs more and better marketing creative based on real pain points, working patterns, and market intelligence.

## Simple overview
Use these modules when the startup needs new creative angles, more ad variation, or a systematic creative testing engine.

The goal is not random idea generation.
The goal is structured creative production tied to hypotheses.

## Core rule
The creative agent should not generate 100 random ads.
It should generate 100 creative variants from a structured matrix based on:
- pain points
- personas
- desired outcomes
- objections
- proof styles
- working historical patterns
- competitor creative intelligence

## Action modules
### 1. Creative angle matrix
- generate angle buckets across pain, outcome, objection, comparison, proof, and persona
- identify which angles are safest, highest-upside, or underused

### 2. 100-creative generation pack
- produce a large structured set of creatives
- organize by persona, pain point, funnel stage, and format

### 3. Winner-cloning pack
- take winning creatives and branch them into adjacent variants
- preserve the useful pattern while refreshing hooks, proof, and format

### 4. Creative refresh pack
- replace tired hooks and stale frames
- refresh without losing the winning strategic core

### 5. Persona-specific creative set
- tailor creatives to end-user, manager, and executive buyers where relevant
- surface conflicts in what each role responds to

### 6. Landing-page alignment pack
- connect ad angle to landing page promise, proof, and CTA
- reduce message mismatch between ad and destination

## Inputs
- customer pain points
- objections and FAQ patterns
- prior winning creatives
- campaign performance data
- landing page / offer context
- [[Competitor Ad Intelligence]]
- customer language from tickets, calls, interviews, or research

## Outputs
- creative matrix
- 100-ad variation library
- top-priority creative tests
- hooks/headlines/body variants
- format suggestions
- winner-clone variants
- landing-page match notes

## Best use cases
- creative testing program needs more breadth
- startup is stuck reusing weak ad ideas
- team wants to scale creative systematically
- need fresh variations grounded in what already works
