# startupClaw - Action Modules - Onboarding & Retention

## Purpose
Define onboarding and retention action modules startupClaw can execute when users/customers are not reaching or repeating value.

## Simple overview
Use these modules when the startup has signal of demand but weak activation, onboarding, or retention.

## Action modules
### 1. Onboarding email sequence draft
- generate first-week onboarding emails
- tie emails to activation milestones

### 2. Getting-started checklist
- generate the clearest first-step path for a new user/customer
- reduce setup friction

### 3. First-week journey map
- map what should happen from signup/purchase to first value
- identify likely drop-off points

### 4. Churn / retention interview script
- generate questions for churned and retained users
- identify repeat-value gaps and expectation mismatch

## Best use cases
- activation bottleneck
- onboarding friction
- first-month churn
- weak repeat usage / repeat purchase
