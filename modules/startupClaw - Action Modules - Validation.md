# startupClaw - Action Modules - Validation

## Purpose
Define validation-oriented action modules startupClaw can execute to reduce startup uncertainty fast.

## Simple overview
Use these modules when the startup needs better evidence for:
- problem validity
- audience fit
- willingness to pay
- strongest use case

## Action modules
### 1. Interview script generator
- generate customer interview questions
- tailor by audience and pain hypothesis

### 2. Willingness-to-pay test script
- generate pricing / value questions
- test willingness to buy, not just interest

### 3. Segment comparison test
- compare which audience reacts strongest
- compare which use case feels most urgent

### 4. Assumption pressure test
- identify which assumptions are still weak
- generate validation prompts for the biggest unknowns

## Best use cases
- problem discovery
- solution validation
- weak evidence quality
- unclear willingness to pay
