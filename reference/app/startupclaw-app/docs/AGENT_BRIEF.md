# Coding Agent Brief

Build startupClaw as an MVP product.

## What to build
A founder-facing web app where a founder enters a short business statement and receives:
- startup report
- audience expansion
- alternative buyer mapping
- stage + bottleneck diagnosis
- one best next move
- one recommended execution agent
- one generated asset

## Required first pages
- landing page
- analyze/input page
- report page
- asset page

## Required first asset generators
- interview script
- landing page draft
- pricing test plan

## Product rule
Bias toward:
- simplicity
- usefulness
- speed
- demo quality

Do not overbuild auth, payments, or collaboration in the first pass.
