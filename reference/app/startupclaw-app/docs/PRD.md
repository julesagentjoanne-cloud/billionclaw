# Product Requirements Document

## Product summary
startupClaw is a founder product that turns a simple business statement into:
- a startup intelligence report
- audience expansion ideas
- alternative buyer mapping
- demand-signal analysis
- stage and bottleneck diagnosis
- one best next move
- optional buyable execution agents that produce useful startup assets

## Primary user
- early-stage founder
- indie hacker
- solo founder
- small startup team

## Core input
Business statement:
- we are building X for Y
- they currently use Z
- they should choose us because A
- they would pay because B
- optional evidence

## Core output
- startup interpretation
- existing demand evidence
- audience expansion
- alternative buyer mapping
- persona + urgency
- revenue proximity
- competitor / alternative map
- stage + bottleneck
- best next move
- recommended execution agent
- what startupClaw did

## MVP scope
### Must-have
- business statement input
- report generation
- audience expansion
- alternative buyer mapping
- stage + bottleneck logic
- one best next move
- one recommended execution agent
- 3 generated asset types
- clean landing page

### Nice-to-have later
- accounts
- saved projects
- payment flow
- review loop dashboard
- public report links
